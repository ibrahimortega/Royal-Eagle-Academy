# Gemini Bourse Buddy

مشروع واجهة مستخدم مبني بـ Vite + React لــ "Gemini Bourse Buddy".

الهدف: تطبيق تعليمي وأدوات مساعدة في التداول والبيانات.

بدء التشغيل (محلي)

1. انسخ المتغيرات المطلوبة إلى ملف `.env` أو استخدم `.env.example` كنقطة بداية.

2. ثبت الحزم ثم شغّل الخادم:

```bash
npm install
npm run dev
```

الأوامر المتاحة:

- `npm run dev` — تشغيل بيئة التطوير.
- `npm run build` — إنتاج البناء في `dist/`.
- `npm run preview` — معاينة نسخة البناء محلياً.
- `npm run test` — تشغيل الاختبارات.

متغيرات بيئة مطلوبة (مثال في `.env.example`):

- `VITE_SUPABASE_URL` — عنوان Supabase.
- `VITE_SUPABASE_PUBLISHABLE_KEY` — مفتاح Supabase (نشر).
- `LOVABLE_API_KEY` — مفتاح واجهة AI المستخدمة في وظائف Supabase (لا تضف المفتاح للمستودع).

ملاحظات أمنية:

- لا تضع مفاتيح أو أسرار حقيقية في المستودع. استخدم `.env` واحتفظ به خارج Git.
- يفضّل فحص تاريخ Git للأسرار باستخدام أدوات مثل `gitleaks` قبل النشر.
- أضفت `.gitignore` لاستبعاد مخرجات البناء وملفات env الحساسة. إن كانت هناك مخرجات مبنية بالفعل في Git (مثل `dist/`)، فاحذفها من التاريخ أو أزلها ثم ادفع التغييرات.

إذا تريد، أستطيع: إنشاء سكربت لإزالة `dist/` من Git history، تشغيل فحص أسرار، أو تحديث صفحة التوثيق بمزيد من التفاصيل.
